Арендуем два самоката Whoosh из одного профиля

27.08.2024

Персональный электротранспорт стал неотъемлемой частью городских пейзажей. Люди используют его для передвижения из пункта А в пункт Б, для прогулок и даже для свиданий. Кстати, для последнего варианта можно использовать 2 самоката Whoosh не только в тех ситуациях, когда у обоих на смартфоне есть приложение сервиса (и деньги для аренды). Для организации "двойной поездки" можно использовать один профиль. Рассказываем вам, как это сделать.

Ну а для начала поездки вам конечно же понадобится хотя бы один рабочий аккаунт. И для его создания разумно использовать виртуальные номера для регистрации в Whoosh от смс-активатора "Grizzly SMS". Они стоят недорого, гарантируют анонимность и дают отличную возможность разблокировать самокат и начать поездку.

Можно ли в Whoosh взять два самоката на один аккаунт?

Если у вашего спутника/спутницы не установлено приложение для аренды электротранспорта (и нет желания/возможности его устанавливать) либо же если приложение есть, но нет подключенных платежных методов с достаточной суммой на счету, то можно арендовать у Whoosh 2 самоката из одной учетной записи. Сделать это несложно, но нужно учесть несколько условий.

На практике это выглядит так:

  1. Вы выбираете первый вушбайк и активируете поездку;
  2. Далее в приложении появляется кнопка "+Взять еще". По ней необходимо тапнуть;
  3. После этого появится возможность взять 2 самоката Whoosh;
  4. Таким же образом можно добавить и третий байк.

Что следует помнить:

  • каждый байк требует отдельного депозита. Убедитесь, что у вас на счету есть нужная сумма;
  • взять более 3 байков не получится;
  • поездки на каждом самокате завершаются независимо друг от друга;
  • владелец аккаунта должен находится рядом с каждым из арендованных байков в момент завершения поездок (именно для их завершения). Поэтому разъехаться не получится.

Кстати, если причина, по которой ваш спутник не желает регистрироваться в приложении для аренды байков – нежелание использовать свой номер телефона для передачи онлайн-сервису, то есть и другой способ устроить групповые покатушки, получив два аккаунта Whoosh. Об этом читайте далее.

Как взять 2 самоката Whoosh анонимно?

Без регистрации аккаунта нельзя получить доступ к функционалу большинства интернет-сервисов. А для многих это неприемлемо, поскольку регистрация требует привязку телефонного номера, без этого не получится арендовать два самоката Whoosh. Если же вы стоите перед выбором сохранить анонимность или проехаться на байке, мы предлагаем компромиссный вариант: использовать для регистрации в сервисе виртуальный номер телефона. Это удобно и надежно. А если используете для этих целей смс-активатор "Grizzly SMS", то еще и сэкономите на получении верификационного кода. Кроме того у нас:

  • гарантия доставки смс;
  • возможность выбора страну мобильного оператора из более, чем 150 вариантов;
  • 200+ сервисов для регистрации учетных записей;
  • удобные способы оплаты;
  • Крутая служба техподдержки 24/7.

Разберем на конкретном примере, как работает наш сервис. Перед тем, как арендовать два самоката Whoosh, сделайте следующее:

  1. Зарегистрируйтесь на нашем сайте;
  2. Выберите сервис для регистрации аккаунта (Whoosh) и страну телефонного оператора (любую доступную);
  3. Проверьте стоимость получения кода из смс и пополните счет с помощью банковской карты, криптовалюты или электронного кошелька одной из предложенных платежных систем;
  4. Подтвердите покупку (кнопка "Получить");
  5. Скопируйте купленный номер;
  6. Скачайте, установите и запустите приложение Whoosh на ваш смартфон (если не сделали этого ранее);
  7. Запустите регистрацию нового профиля;
  8. Укажите купленный номер в качестве личного и запросите прием смс;
  9. Вернитесь в кабинет на сайте "Grizzly SMS" и дождитесь получения верификационного сообщения;
  10. Используйте полученный код для завершения регистрации.

После этого можно привязать к созданному профилю платежный метод и взять 2 самоката Whoosh с одного аккаунта.
