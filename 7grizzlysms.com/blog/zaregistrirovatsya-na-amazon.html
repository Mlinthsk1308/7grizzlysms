Как зарегистрироваться на Амазон без номера телефона

01.11.2021

Сайт Амазон является одной из крупнейших торговых площадок в мире. За 2024 год здесь проведено несколько миллиардов сделок. Не составит труда найти товар под любые запросы. Особенно привлекательными для российских покупателей являются условия проведения акций. Продавцы могут реализовывать товар со скидками 70-80%. Это делает покупку выгодной, даже с учетом пересылки на другой континент. Ищите, как зарегистрироваться на Amazon без номера телефона? Следует всесторонне разобраться в данном вопросе.

Можно ли создать аккаунт на Amazon без номера телефона

Торговая площадка ответственно относится к сбору данных о потенциальных покупателей. Процесс регистрации реализован таким образом, чтобы усложнить задачу создания фейковых профилей. При создании учетной записи необходимо ввести мобильный номер или электронную почту. При этом, не допускается использование российского телефона. Подобный фактор не станет препятствием на пути отправки СМС. Таким образом, создать аккаунт Амазон без номера телефона предполагает ввод адреса электронной почты. Подобный подход не избавит от проблем. Торговая площадка продолжит регулярно предлагать добавить мобильный телефон.

 

Рисунок 1 Регистрация Амазон без номера

Не стоит пытаться купить сим-карту или уже существующую учетную запись. Для регистрации профиля можно использовать виртуальный номер. Далее предоставим инструкцию по его приобретению, а также созданию аккаунта на торговой площадке.

Как зарегистрироваться на Амазон без номера телефона с помощью «Grizzly Sms»

Существует большое количество сервисов, которые занимаются продажей виртуальных номеров. Сайт «Grizzly Sms» предлагает оптимальные условия для клиентов. Следует отметить низкие цены, всегда рабочие виртуальные номера для Амазон, простоту покупки и многое другое. Опишем процедуру приобретения:

  1. Создаем собственный аккаунт. Из личных данных требуется указать только адрес электронной почты.

Рисунок 2 Как завести учетную запись Амазон без номера телефона

2. Для совершения покупок пополняется счет в личном кабинете.

Переходим к наиболее важной части. Требуется купить виртуальный номер для регистрации на Amazon (Амазон). Сначала выбираем страну, потом указываем сервис. Нажимаем кнопку «Купить».

Рисунок 4 Купить виртуальный номер для регистрации на Amazon (Амазон)

Полученный виртуальный номер может использоваться для приема СМС с кодом активации с сервиса Амазон. Его можно найти в разделе учетной записи «Активные номера».

Как зарегистрировать аккаунт Амазон

Если говорить о подготовке, то Вам потребуется адрес в США, куда будет происходить доставка. Торговая площадка не предусматривает выгодных условий транспортировки товаров в Россию. Некоторые продавцы вообще не предлагают подобной услуги. Существуют специальные сервисы, среди которых наиболее популярными считаются:

  • Бандеролька;
  • Полярный экспресс;
  • Shipito;
  • Pochtoy com;
  • Shopfans.

Теперь можно переходить к рассмотрению инструкции по регистрации на Amazon без номера телефона (привязанного к сим-карте).

  1. Переходим на страницу входа в аккаунт. Нажимаем на кнопку в нижней части «Create your Amazon account».

Рисунок 5 Как зарегистрировать Амазон без номера: пошаговая инструкция

2. Мы попадаем на страницу регистрации учетной записи. Вводим имя, которое будет использоваться на сайте, а также почту и пароль для нового профиля. Вводим купленный ранее виртуальный номер. После нажатия «Continue», отправляется СМС с кодом активации. Его можно получить в личном кабинете на сайте «Grizzly Sms». Вводим в соответствующее поле и продолжаем регистрацию.

Рисунок 6 Регистрация учетной записи Амазон без номера телефона

3. На данном этапе необходимо указать адрес доставки посылок на территории США. Здесь нам помогут сервисы, которые указаны ранее. Там будет подробная информация с инструкцией, как заполнить все поля.

Рисунок 7 Как создать Amazon (Амазон) без номера

4. В следующем окне пользователю предлагают проверить введенную информацию и исправить ее (если в этом есть необходимость). Оставляем оригинальный адрес и продолжаем регистрацию учетной записи.

Рисунок 8 Регистрация Amazon без номера телефона

5. Указываем способ доставки. Тут все зависит от сроков, если время не играет первостепенной роли и хочется сэкономить, оставляем «Standard».

Рисунок 9 Как зарегистрировать аккаунт Amazon без номера телефона

На этом регистрация аккаунта на Amazon (Амазон) без номера телефона завершена. Можно заказывать товар у продавцов.
